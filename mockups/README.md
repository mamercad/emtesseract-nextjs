## Design Features

### Visual Improvements
- **Glass-morphism**: Frosted glass effect cards with backdrop blur
- **Better Typography**: Space Grotesk for headings, Inter for body text
- **Gradient Accents**: Purple-to-indigo gradients for CTAs and highlights
- **Subtle Glows**: Soft box shadows on interactive elements
- **More Whitespace**: Generous padding and spacing for better readability

### Interactive Elements
- Smooth hover transitions with elevation changes
- Animated floating tesseract logo
- Scroll indicator on hero section
- Enhanced card interactions

### Color Palette
- Primary: #6366f1 (Indigo)
- Secondary: #8b5cf6 (Purple)
- Background: #0a0a0f (Deep dark)
- Text: #e5e7eb (Light gray)
- Accents: Radial gradients with primary/secondary colors
